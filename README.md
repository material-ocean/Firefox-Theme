### Installation:

- Add [Firefox-Color](https://addons.mozilla.org/en-US/firefox/addon/firefox-color/) Addon to your firefox browser
- [Open the theme link](https://kutt.it/367yhv) via your browser
- Select Yup to apply the theme
- Enjoy!
